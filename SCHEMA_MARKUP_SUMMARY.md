## Maintenance

### When to Update Schema:
1. **Property Listings** - Update when properties are added/removed
2. **Market Reports** - Update dateModified when reports are refreshed
3. **Reviews** - Update AggregateRating when new reviews are added
4. **Amenities** - Update ItemList when amenities change
5. **Schools** - Update ItemList when school information changes

### Best Practices:
- Keep schema data accurate and up-to-date
- Use appropriate schema types for content

- Test schema after major updates
